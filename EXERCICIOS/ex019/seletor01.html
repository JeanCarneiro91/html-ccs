Criando Sites com HTML e CCS

Aprendendo HTML

HTML Básico

Nesta etapa, você aprende a estrutura fundamental das páginas web. O foco está nas principais tags como <html>, <head>, <body>, além de elementos como títulos (<h1> a <h6>), parágrafos (<p>), listas (<ul> e <ol>), links (<a>), e imagens (<img>). Também é ensinado como criar tabelas e formulários simples com <table> e <form>. O objetivo é entender a construção e organização básica de uma página.

HTML Intermediário

Aqui você passa a trabalhar com conceitos mais estruturados, como formulários complexos (com validação, <input> de diferentes tipos e <select>), semântica HTML5 (tags como <article>, <section>, <nav> e <aside>). Também se aprende a inserir mídia (áudio e vídeo), além de trabalhar com meta tags no <head> para SEO e responsividade.

HTML Avançado

O HTML Avançado envolve boas práticas de acessibilidade (usando atributos ARIA e labels adequados), SEO otimizado, integração com frameworks (como Bootstrap) e uso de APIs para inserir conteúdo dinâmico. Além disso, você aprende a estruturar layouts mais sofisticados com iframes e utilizar templates reutilizáveis com Web Components.

Aprendendo CSS

CSS Básico

Aqui você aprende como estilizar elementos HTML com cores, tamanhos de texto, margens e padding. O foco está em entender seletores básicos (como p, .classe, #id) e propriedades simples (como color, font-size, background-color). Também é introduzido o conceito de folhas de estilo externas e inline styles.

CSS Intermediário

Neste nível, você trabalha com layouts mais complexos utilizando Flexbox e Grid, aprende sobre media queries para design responsivo e trabalha com pseudo-classes e pseudo-elementos (ex: :hover, ::before). Outras técnicas incluem transições, animações simples e sombras com box-shadow.

CSS Avançado

No nível avançado, você domina animações complexas com @keyframes, cria layouts totalmente responsivos e fluídos, e integra pré-processadores como SASS ou LESS. Também se aprende CSS modular e design system para manter projetos escaláveis e consistentes. O uso de variáveis CSS e técnicas de otimização de performance também faz parte desse estágio.
